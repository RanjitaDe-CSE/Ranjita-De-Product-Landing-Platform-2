Terracotta Product Landing Page 
Visit Site: https://ranjitade-cse.github.io/Ranjita-De-Product-Landing-Platform-2/
Project Overview
This project is a responsive Product Landing Page developed using HTML5 and CSS3. It is designed to showcase terracotta products such as home décor, jewellery, and kitchen utensils in a visually appealing and structured layout.

The main objective of this project is to demonstrate a clear understanding of the CSS Box Model, including the use of margin, padding, borders, and element sizing (width and height).

Objectives
- Apply CSS Box Model concepts effectively
- Create a clean and readable layout
- Improve frontend styling and design skills
- Build a responsive and user-friendly webpage


 Features
- Responsive design using media queries
- Clean and minimal UI design
- Proper use of CSS Box Model:
  - Padding for internal spacing
  - Margin for external spacing between sections
  - Borders to define containers
  - Border-radius for rounded corners
  - Width & Height for layout control
- Product image with description
- Feature cards with bordered containers
- Testimonials section with styled boxes
- Call-to-action button ("Buy Now")

Technologies Used
- HTML5
- CSS3
- Google Fonts (Poppins)
